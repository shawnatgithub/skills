# 系统思维（Systems Thinking）

## 作用

识别反馈循环（feedback loops）、涌现属性（emergent properties）、杠杆点（leverage points）和系统边界。

## 核心原则

1. **整体大于部分之和**：系统的行为不能仅通过理解其组成部分来预测
2. **因果是循环的，不是线性的**：A影响B，B也反过来影响A
3. **行为由结构决定**：要改变行为，先改变系统结构
4. **延迟无处不在**：行动和结果之间存在时间差
5. **增长有极限**：任何指数增长最终都会遇到约束

## 执行步骤

### 步骤 1：界定系统边界

明确什么在系统内，什么在系统外。

**关键问题**：
- 我们关注的系统是什么？它的边界在哪里？
- 哪些要素是系统内部的？哪些是外部环境？
- 系统的目标/功能是什么？

**输出格式**：
```
【系统边界定义】
- 系统名称：___
- 系统目标/功能：___
- 系统边界：包含___，不包含___
- 时间范围：___
```

### 步骤 2：识别核心要素

列出系统中的关键要素（变量、角色、资源等）。

**输出格式**：
```
【核心要素清单】
1. 要素A：___（类型：存量/流量/角色/资源）
2. 要素B：___（类型：存量/流量/角色/资源）
3. 要素C：___（类型：存量/流量/角色/资源）
...
```

### 步骤 3：绘制反馈循环

识别要素之间的因果关系，找出增强回路和调节回路。

**两种反馈循环**：

| 类型 | 特征 | 例子 |
|------|------|------|
| **增强回路（Reinforcing）** | 越多越多，越少越少。指数增长或加速衰退 | 复利、口碑传播、贫困循环 |
| **调节回路（Balancing）** | 趋向目标，维持稳定。有目标状态和纠正机制 | 体温调节、供需平衡、库存管理 |

**输出格式**：
```
【反馈循环分析】
增强回路：
- 回路1：A→B→C→A（描述：___）
- 回路2：___

调节回路：
- 回路1：___→目标→___（描述：___）
- 回路2：___
```

### 步骤 4：识别延迟

找出行动与结果之间的时间延迟。

**常见延迟点**：
- 感知延迟：发现问题需要时间
- 决策延迟：做出决策需要时间
- 实施延迟：执行到位需要时间
- 效果延迟：看到结果需要时间

**输出格式**：
```
【关键延迟】
1. 延迟点：___ → 延迟时长：___
2. 延迟点：___ → 延迟时长：___
3. 延迟点：___ → 延迟时长：___
```

### 步骤 5：发现杠杆点

找到系统中"四两拨千斤"的位置——小改变带来大效果。

**杠杆点排序（从弱到强）**：
1. 数值参数（税率、价格、配额）
2. 存量与流量的结构
3. 时间延迟的长度
4. 负反馈回路的强度
5. 正反馈回路的增益
6. 信息流动结构
7. 系统规则（激励、惩罚、约束）
8. 自组织能力
9. 系统目标
10. 心智模型/范式

**输出格式**：
```
【杠杆点分析】
- 高杠杆点（小改变→大效果）：
  1. ___
  2. ___
- 低杠杆点（努力很多，效果有限）：
  1. ___
  2. ___
```

### 步骤 6：预测系统行为

基于系统结构，推演系统可能的行为模式。

**常见行为模式**：
- 指数增长（增强回路主导）
- 目标追寻（调节回路主导）
- S型增长（增强+调节，有上限）
- 振荡（调节回路+延迟）
- 转移负担（饮鸩止渴的结构）

**输出格式**：
```
【系统行为预测】
- 短期（___）：___
- 中期（___）：___
- 长期（___）：___
- 关键转折点：___
```

## 适用场景

| 场景 | 为什么适合 |
|------|-----------|
| 组织/管理问题 | 组织是复杂系统，简单因果往往失效 |
| 生态/市场分析 | 多变量互动，反馈循环无处不在 |
| 政策/社会影响 | 政策干预的间接效果和长远影响 |
| 复杂故障排查 | 故障不是单点问题，而是系统失效 |
| 战略规划 | 理解系统结构，找到高杠杆的战略切入点 |

## 常见陷阱

1. **边界画错**：系统边界太窄或太宽，导致分析失真
2. **只看事件，不看结构**：停留在"谁做了什么"，没看到深层的系统结构
3. **线性思维**：用简单因果解释复杂系统行为
4. **忽视延迟**：因为没有立刻看到效果，就认为干预无效，然后过度干预
5. **就事论事**：只处理症状，不改变产生问题的系统结构

## 与其他工具的配合

- **second-order-effects**：系统思维看整体结构，第二序效应看连锁后果
- **first-principles-thinking**：第一性原理拆解要素，系统思维看要素间的关系
- **chain-of-thought**：用链式思考逐步推演系统行为的具体路径
