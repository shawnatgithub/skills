# 苏格拉底式质疑 + 反转思维（Socratic + Inversion）

## 作用

通过连续提问挑战假设、揭示隐藏信念；通过反向思考打破认知盲区。

## 第一部分：苏格拉底式质疑

### 核心原则

1. **不直接给出答案**，而是通过提问引导对方自己发现答案
2. **质疑"理所当然"**：对看似不言自明的说法追问依据
3. **逐层深入**：每个回答都成为下一个问题的起点
4. **保持好奇，而非挑衅**：目的是探索真理，不是赢得辩论

### 经典提问框架

#### 1. 澄清性提问（Clarification）
确保真正理解对方在说什么。

- "你说的___具体是什么意思？"
- "能否举个例子？"
- "这个概念能用更简单的话解释吗？"

#### 2. 探询假设（Probing Assumptions）
检查未明说的前提。

- "你在这里假设了什么？"
- "如果这个假设不成立，会怎样？"
- "你怎么知道这个假设是对的？"

#### 3. 探询理由和证据（Probing Reasons and Evidence）
要求提供支撑观点的依据。

- "是什么让你相信这一点？"
- "有什么证据支持这个说法？"
- "反例呢？有没有不支持这个说法的情况？"

#### 4. 探询视角（Probing Perspectives）
考虑其他可能的观点。

- "反对者会怎么说？"
- "从另一个角度看呢？"
- "如果站在___的立场，会怎么看？"

#### 5. 探询后果和影响（Probing Implications）
推演如果这个观点成立，会导致什么。

- "如果这是真的，那又会怎样？"
- "这会带来什么影响？"
- "长期来看呢？"

#### 6. 质疑问题本身（Questioning the Question）
反思问题本身是否成立。

- "为什么这个问题重要？"
- "有没有更好的问题？"
- "这个问题是不是问错了？"

   - 回答/反思：___

- 原观点中站得住脚的部分：___

## 第二部分：反转思维

### 核心原则

1. **反过来想**：与其想"如何成功"，不如想"如何确保失败"
2. **避免愚蠢比追求聪明更容易**：很多时候，不犯傻比做对事更重要
3. **识别失败模式**：清楚知道什么会导致失败，然后避免那些做法

### 反转提问框架

#### 1. 目标反转
"如果我想达成完全相反的目标，我会怎么做？"

- 原目标：___
- 反转目标：___
- 如何达成反转目标：___
- → 因此，我应该避免：___

#### 2. 假设反转
"如果我坚信的这个假设是错的，会怎样？"

- 原假设：___
- 反转假设：___
- 如果反转假设成立，会怎样：___
- → 我需要检验：___

#### 3. 因果反转
"如果结果是原因，原因是结果呢？"

- 原因果：A导致B
- 反转：B导致A
- 反转是否成立：___
- → 真正的关系可能是：___

#### 4. 视角反转
"如果我是对方/对手/受害者，我会怎么想？"

- 我的视角：___
- 对方视角：___
- 差异在哪：___
- → 我忽略了：___

### 输出格式

```
【反转思维分析】
分析主题：___

1. 目标反转：
   - 原目标：___
   - 如何达成相反目标：___
   - 因此应该避免：___

2. 假设反转：
   - 核心假设：___
   - 如果假设不成立：___
   - 需要验证：___

3. 视角反转：
   - 我方视角：___
   - 对方视角：___
   - 盲点：___

【反转揭示的风险/盲区】
1. ___
2. ___
3. ___

【应对策略】
1. ___
2. ___
3. ___
```

## 适用场景

| 场景 | 为什么适合 |
|------|-----------|
| 决策质量检验 | 用质疑和反转找出决策中的漏洞 |
| 风险评估 | 主动寻找可能的失败模式 |
| 创新验证 | 用反面思考检验新想法的鲁棒性 |
| 认知盲区挖掘 | 发现自己没意识到的偏见和盲点 |
| 辩论/谈判准备 | 预判对方可能的质疑和反对理由 |

## 常见陷阱

1. **杠精化**：为了质疑而质疑，失去建设性
2. **只破不立**：找出了一堆问题，但没有提出替代方案
3. **相对主义**：质疑一切，最后什么都不相信
4. **反转滥用**：不是所有问题都适合反转，有些反转是抬杠

## 与其他工具的配合

- **first-principles-thinking**：先用苏格拉底式质疑挑战假设，再用第一性原理重建
- **chain-of-thought**：用苏格拉底式质疑检验链式推理的每一步
- **second-order-effects**：反转思维看直接后果的反面，第二序效应看长远连锁反应
